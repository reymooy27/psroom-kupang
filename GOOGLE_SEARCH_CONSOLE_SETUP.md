# 🔍 GOOGLE SEARCH CONSOLE SETUP GUIDE

## PS Room Kupang - Search Console Implementation

---

## 1️⃣ **BUAT SEARCH CONSOLE ACCOUNT**

### Step 1: Sign Up
1. Buka https://search.google.com/search-console
2. Login dengan Google account
3. Klik **"Add property"**

### Step 2: Pilih Property Type

**Pilih: URL prefix** (lebih mudah untuk GitHub Pages)

```
URL: https://reymooy27.github.io/psroom-kupang/
```

Klik **Continue**

---

## 2️⃣ **VERIFIKASI PROPERTY**

### Method 1: HTML File Upload (Recommended untuk GitHub Pages)

1. **Download HTML file** dari Search Console
2. **Upload ke GitHub repo:**
   - Nama file: `googleXXXXXXXXXXXX.html` (sesuai dari Google)
   - Location: Root folder
3. **Commit & Push**
4. **Verify** di Search Console

### Method 2: HTML Tag (Lebih Mudah)

1. Pilih **"HTML tag"**
2. Copy meta tag:
   ```html
   <meta name="google-site-verification" content=XXXXXXXXXXXXXXXXXXXXX />
   ```
3. **Paste di `<head>` website** (setelah `<title>`)
4. **Commit & Push**
5. **Verify** di Search Console

### Method 3: DNS Verification (Advanced)

1. Pilih **"Domain name"**
2. Add TXT record di DNS provider
3. Wait 24-48 hours
4. Verify

---

## 3️⃣ **SUBMIT SITEMAP**

Setelah verified:

1. **Menu:** Sitemaps (sidebar kiri)
2. **Enter sitemap URL:** `sitemap.xml`
3. **Klik:** Submit
